# 📘 BOOMER OFF Vintage 网页版品牌手册

## 🎨 设计语言：复古昭和 × 红黑撞色

借鉴日本昭和年代海报与中古杂货铺美学，呼应 BOOMER OFF Logo 的视觉基因：

- **主色调**：BOOMER 红 `#E60012`（取自 Logo）+ 深炭黑 `#1A1A1A` + 米黄底纸 `#F5EFE0`
- **辅助色**：复古奶白、做旧金、杂志墨绿点缀
- **字体搭配**：
  - 中文标题：思源宋体 / 站酷文艺体（带昭和印刷质感）
  - 中文正文：思源黑体
  - 英文标题：Anton / Bebas Neue（呼应 Logo 圆润粗体）
  - 装饰手写：Caveat / Permanent Marker（呼应 "Vintage" 手写字）
- **装饰元素**：网点底纹、做旧纸张纹理、复古印章、胶带贴纸、佐藤象插画剪影、唱片机/Gameboy 像素icon、虚线分割、红色高亮笔涂抹

## 🖥️ 网页交互体验

### 浏览方式
- **横屏 16:9 幻灯片** 全屏展示，居中自适应缩放（1920×1080 渲染基准）

## ⚙️ 技术实现要点

### 网页端
- React + Tailwind 构建可缩放幻灯片系统（每页一个 React 组件）
- 1920×1080 固定基准 + CSS scale 自适应，保证任意屏幕一致呈现
- 路由：`/` 为首页 PPT 浏览器；URL 带 `#slide-7` 锚点支持深度链接
- 键盘 / 触摸 / 点击全交互方案
- 暗色模式预留切换（默认昭和米黄底）

### 下载功能
- **下载 PPTX**：使用 `pptxgenjs` 在浏览器端生成 .pptx 文件，每页对应一张幻灯片，文字、配色、布局与网页版高度一致
- **下载 PDF**：使用 `html2canvas` + `jsPDF` 将每张幻灯片渲染为 PDF 页面，导出 16:9 横版 PDF
- 两种文件均在用户点击时实时生成并下载，无需后端

### Logo 资源
- 上传的 Logo 文件保存到 `src/assets/boomer-off-logo.png`，作为 ES 模块导入，封面页与每页页眉 / 页脚使用
