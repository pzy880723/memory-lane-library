## 优化方案

根据您的三点反馈，我将对品牌手册进行三大维度升级。

---

### 🎨 一、配色调整 — 去黑、保留红米双主色

**现有问题**：4 张幻灯片（章节扉页、内容生产机器、翻筐乐、对比表头）使用纯黑 `#1A1A1A` 背景，整体偏沉重。

**优化方向**：
- **替换所有黑色背景**为以下三种之一：
  - **米黄底纸（主色）** `#F5EFE0` — 70% 的页面用作主背景
  - **BOOMER 红（强调色）** `#E60012` — 章节扉页、金句页、视觉锚点
  - **复古暖色辅助** — 引入 `vintage-gold (赭石黄)` 与 `vintage-cream-deep (奶油深米)` 做层次
- **保留黑色**：仅作为文字、细描边、点缀使用（如小标签、印章），不再作为大面积背景
- **章节扉页**：从「黑底+大字」改为「红底+米黄镂空大字」+ 昭和波点拼贴
- **更新 SlideShell**：`variant="ink"` 改为 `variant="red"` 或 `variant="cream-deep"`

---

### 💥 二、排版活力升级 — 解决"死板 + 字小"问题

**现有问题**：
- 大量页面采用规整的 3×2 / 4×1 网格，节奏单一
- 正文常用 `text-base / text-lg`（缩放后 ≈ 16-20px）在 1920px 画布上偏小
- 副标题、说明文字层级不够明显

**优化方向**：

**1. 字号全面放大**（建立新的"幻灯片字号阶梯"）
| 用途 | 现有 | 优化后 |
|------|------|--------|
| 正文 | text-lg / text-xl (18-20px) | **text-2xl / text-3xl (24-30px)** |
| 卡片说明 | text-base | **text-xl** |
| 副标题 | text-2xl | **text-3xl / text-4xl** |
| 大数字 | text-7xl | **text-8xl / text-9xl** |

**2. 排版破格设计**（让画面跳起来）
- **倾斜元素**：标题、徽章、手写字加 `rotate(-2deg ~ 3deg)` 营造拼贴感
- **大色块切割**：用红色斜带、米色块对角分割，打破方格感
- **数字戏剧化**：核心数据（300万+、10万+）用 `text-[16rem]` 超大字 + 红色描边
- **重叠层次**：胶带、印章、贴纸、手写批注叠在卡片上方
- **不对称网格**：用 `grid-cols-12 + col-span` 做 5/7、4/8、3/9 切分，避免均分
- **波点 / 网线 / 划痕 纹理**作为背景填充，增加昭和漫画感

**3. 内容层级强化**
- 每页设立一个"焦点元素"（巨型数字 / 大字标题 / 主图），其他元素围绕它布局
- 标题使用 `highlight-red` 高亮笔涂抹效果增加节奏
